# Subnet Calculator

A Python CLI tool to calculate network, broadcast, and host range from any IP/CIDR.
Built to practice networking concepts and Python fundamentals.

## Features
- Calculate Network Address
- Calculate Broadcast Address 
- Calculate Subnet Mask
- Show Total and Usable Hosts
- Show Usable IP Range

## Usage
```bash
python main.py
Enter IP in CIDR format:
192.168.1.0/24
Network Address: 192.168.1.0
Broadcast Address: 192.168.1.255
Subnet Mask: 255.255.255.0
Total Hosts: 256
Usable Hosts: 254
Usable Range: 192.168.1.1 - 192.168.1.254
